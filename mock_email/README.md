# Mock Email Integration Without GoogleMCP

Connect MARVIN to mock_email.

## What It Does

- **Mock** - Read email and create drafts in the mock_email folder

## Who It's For

Anyone who loads email messages into the mock_email folder for work or personal use.

## Prerequisites

- Ability to run Claude Code
- Have YPS app set up to run

## Setup

1. Copy email messages into md files in the mock_email/Inbox folder
2. The script will check for messages in this location
3. Drafts will be loaded into the mock_email/Drafts folder

## Try It

After setup, try these commands with MARVIN:

- "Show me my unread emails"

## Danger Zone

No email messages can be sent from mock. If you need to send messages, this is not the way to do so.
